@@ All basic commands included in the bot
@@ Voice commands should be called after saying 'tony'
@@ 'off' anywhere in the voice command (no 'tony' needed) will turn the bot off
Mouse:
    1) 'mouse north/south/east/west <dist>' : move mouse up/down/right/left <dist>
    2) 'scroll up/down <repeat>' : scroll up/down <repeat> times
    3) 'drag north/south/east/west <dist>' : drag (hold mouse down)
    4) 'left/right click <repeat>' : left/right click <repeat> times (default is 1)
Keyboard:
    5) 'press <key>' : press a single key
    6) 'type <text>': type <text> even if impossible
    7) 'combine <keys>' : press all <keys> in combination
Misc:
    8) 'web <site>': launch <site> on the default browser. No 'www' please.
        e.g. web youtube.com    
    9) 'minimize/maximize' : minimize/maximize the current window (only for windows)
    10)'search <query>' : search for <query> on Google. Depends on connection.
